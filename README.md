# Investigating-Netflix-Movies-Exploratory-Data-Analysis-of-1990s-Films-on-Netflix
This project analyzes Netflix catalog data to understand characteristics of movies released in the 1990s, supporting a production company interested in nostalgic content. Using the netflix_data.csv dataset, it explores fields such as title, director, cast, country, release year, duration, description, and genre to answer questions like typical movie length in the decade and how many short action movies exist.
