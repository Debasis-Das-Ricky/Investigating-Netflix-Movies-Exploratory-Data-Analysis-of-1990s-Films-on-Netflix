# Investigating-Netflix-Movies-Exploratory-Data-Analysis-of-1990s-Films-on-Netflix
Exploratory Data Analysis of 1990s titles using the netflixdata.csv dataset (show id, type, title, director, cast, country, date added, release year, duration, description, genre) to uncover patterns in movie durations, genres, and short action films for a nostalgia-focused production company.
